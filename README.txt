── LOGO ────────────────────────────────────────────────────────

  Folder : images/
  File   : logo.png

  Drop your logo file here:
    images/logo.png

  It will automatically appear in:
    • Navigation bar (top left)
    • Hero section (centre)
    • Footer — Home page
    • Footer — Products page

  NOTE: Your current logo.png (from the project root) has already
  been copied to images/logo.png for you. If you update your logo,
  replace images/logo.png — the root logo.png is no longer used.

── SETTING PRICES ──────────────────────────────────────────────

  Open  products.js  and find the product you want to price.
  Change  price: ""  to  price: 89.99  (use a plain number, no $).

  Example:
    price: ""        ← not set yet (shows "—" on the card)
    price: 89.99     ← will display as "$89.99"

  Save the file and reload the page — done.


── TIPS ────────────────────────────────────────────────────────

  • Recommended image size: 800 × 1000 px (portrait), under 300 KB
  • All filenames are lowercase with hyphens — no spaces, no capitals
  • JPG gives smaller file sizes; PNG is fine for transparent logos
  • After uploading all images, reload index.html in your browser
